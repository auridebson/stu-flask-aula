Aula de Flask

Flask é um framework web para Python que permite criar aplicativos web de forma rápida e simples. Ele é conhecido por ser leve, flexível e fácil de usar, proporcionando ao desenvolvedor liberdade para escolher as ferramentas e bibliotecas que deseja usar, sem impor uma estrutura rígida.

A principal filosofia por trás do Flask é "micro", o que significa que ele fornece apenas o essencial para começar a construir aplicativos web, deixando muitas decisões de design e escolhas de ferramentas para o desenvolvedor. Isso contrasta com outros frameworks web mais robustos, como Django, que oferecem mais funcionalidades e uma estrutura mais opinativa.
